## Homework: วิเคราะห์ Danger / Accident / Near Miss
จาก Hazard ที่กำหนด ให้คุณวิเคราะห์ 3 มิติ ได้แก่
- Danger (อันตรายที่อาจเกิดขึ้น)
- Accident (เหตุการณ์ที่เกิดขึ้นจริง)
- Near Miss (เหตุเฉียดที่เกือบเกิด แต่ยังไม่เกิด)
ให้เขียนลงในตารางที่กำหนด โดยใช้ภาษาของตัวเอง ห้ามท่องนิยาม ห้ามคัดลอกทฤษฎี ให้เขียนแบบภาษาคนทั่วไป เหมือนเล่าเหตุการณ์ให้เพื่อนฟัง
สามารถส่งตรวจคราวเดียวทั้งหมดหรือครั้งละหมวดก็ได้
คุณสามารถส่งคำตอบใหม่ได้เรื่อยๆ จนกว่าจะพอใจคะแนน TA AI จะตรวจให้ทุกครั้งแบบเข้มงวด

## แนวทางการเขียน Danger, Accident และ Near Miss
### 👉 วิธีเขียน Danger ให้ถูกต้อง
Danger = “โอกาสที่จะเกิดเหตุ” ต้องมี **สถานการณ์เฉพาะเจาะจง** เช่น
- ตอนเดินถือของร้อน
- ตอนเดินสวนกันในครัวแคบ
- ตอนรีบทำงาน
ห้ามเขียนว่า “มีโอกาสลื่นล้ม” เฉย ๆ เพราะมันสั้นเกินไป ต้องระบุสถานการณ์ที่เจาะจงด้วย

### 👉 วิธีเขียน Accident ให้ถูกต้อง
Accident = “เหตุการณ์จริงที่เกิดขึ้นแล้ว” ต้องมี **ลำดับเหตุการณ์** เช่น
1. เหยียบน้ำมัน
2. ลื่นล้ม
3. มือกระแทกกระทะร้อน
4. อาหารหก
ห้ามเขียนว่า “ลื่นล้มเพราะเดินไม่ระวัง” เพราะเป็นการโทษคน และไม่ใช่หลัก OHS

### 👉 วิธีเขียน Near Miss ให้ถูกต้อง
Near Miss = “เกือบเกิด แต่ไม่เกิด” ต้องมี **สิ่งที่ช่วยให้ไม่เกิดเหตุ** เช่น
- จับโต๊ะทัน
- ยกเท้าหลบทัน
- เพื่อนดึงตัวไว้
ห้ามเขียนว่า “เกือบล้ม” เฉย ๆ ต้องบอกว่า “เพราะอะไรถึงไม่ล้ม”

### K1. พื้นครัวมันลื่นเพราะน้ำมันกระเด็นตอนผัดไฟแรง
#### คำตอบ (ตัวอย่าง)

| รายการ    | รายละเอียด                                                                                                                                    |
| --------- | --------------------------------------------------------------------------------------------------------------------------------------------- |
| Danger    | พนักงานมีโอกาสลื่นล้มตอนเดินถือจานร้อน โดยเฉพาะช่วงที่ครัวแคบและต้องเดินสวนกัน เพราะพื้นมีน้ำมันเคลือบอยู่ทำให้เท้าไถลง่าย                    |
| Accident  | พนักงานเดินถือชามร้อนแล้วเหยียบน้ำมัน ทำให้ลื่นล้ม มือกระแทกกระทะร้อนจนพอง และอาหารหกกระจายเต็มพื้น เกิดความเสียหายทั้งต่อร่างกายและงานในครัว |
| Near Miss | พนักงานเดินผ่านจุดที่มีน้ำมันแล้วรู้สึกเท้าไถล แต่จับโต๊ะทันเลยไม่ล้ม เหตุการณ์นี้เกือบจะกลายเป็นอุบัติเหตุจริง แต่โชคดีที่ควบคุมตัวเองได้ทัน |

### K2. มีดวางคว่ำปลายชี้ออกนอกโต๊ะใกล้ทางเดิน
#### คำตอบ
| รายการ    | รายละเอียด |
| --------- | ---------- |
| Danger    | พนักงานมีโอกาสเดินชนหรือเกี่ยวโดนมีดบาด ตอนที่กำลังรีบเดินยกของผ่านช่องทางเดินแคบ ๆ เพราะปลายมีดยื่นล้ำออกมาในพื้นที่สัญจร |
| Accident  | พนักงานเดินยกตะกร้าผักบังสายตา พอเดินผ่านโต๊ะเตรียมอาหาร สะโพกไปกระแทกด้ามมีด ทำให้มีดร่วงลงมาปักใส่หลังเท้า เลือดไหลอาบ ต้องพาส่งโรงพยาบาล |
| Near Miss | พนักงานเดินเฉียดมุมโต๊ะจนชายเสื้อไปเกี่ยวปลายมีดกำลังจะร่วง แต่โชคดีที่เอามือคว้าด้ามมีดกลางอากาศไว้ได้ทันก่อนที่มันจะตกใส่เท้า |
### K3. ถังแก๊สตั้งชิดเตาไฟจนผิวถังร้อน
#### คำตอบ
| รายการ    | รายละเอียด |
| --------- | ---------- |
| Danger    | ถังแก๊สมีโอกาสเกิดแรงดันสูงจนระเบิดหรือรั่วไหลได้ ตอนที่เชฟเปิดเตาเร่งไฟแรงเพื่อทำเมนูผัดต่อเนื่องนาน ๆ เพราะความร้อนแผ่ไปสะสมที่ตัวถังโดยตรง |
| Accident  | เชฟเปิดเตาต้มน้ำซุปทิ้งไว้ ความร้อนสะสมทำให้วาล์วถังแก๊สที่อยู่ติดกันเสื่อมสภาพ แก๊สเริ่มรั่วออกมาเจอกับประกายไฟจากเตา เกิดไฟลุกพรึบคลอกแขนเชฟและเตาไหม้พังเสียหาย |
| Near Miss | เชฟได้กลิ่นไหม้แปลก ๆ เลยหันไปจับถังแก๊สดูพบว่าร้อนจัดจนมือพอง เลยรีบปิดหัวแก๊สแล้วเอาผ้าชุบน้ำเย็นมาประคบระบายความร้อนได้ทันก่อนแก๊สจะรั่ว |
### K4. หม้อร้อนตั้งบนโต๊ะที่โยกเยก
#### คำตอบ
| รายการ    | รายละเอียด |
| --------- | ---------- |
| Danger    | พนักงานหรือลูกค้ามีโอกาสโดนน้ำซุปร้อน ๆ ลวกตัว ตอนที่เอื้อมมือไปตักอาหารหรือวางจานเพิ่มลงบนโต๊ะ เพราะแรงกดทำให้โต๊ะเอียงจนหม้อสไลด์หลุด |
| Accident  | ลูกค้าใช้ช้อนคนน้ำซุปในหม้อต้ม แล้วจังหวะนั้นขาร้านกดน้ำหนักลงโต๊ะที่โยกเยกพอดี โต๊ะเลยพลิกคว่ำ หม้อน้ำซุปร้อน ๆ ราดใส่ตักลูกค้า ผิวหนังพองเป็นแผลพุพอง |
| Near Miss | พนักงานวางจานข้าวลงบนโต๊ะตัวนั้น ทำให้โต๊ะเอียงวูบ หม้อซุปร้อน ๆ กำลังจะไถลตกพื้น แต่พนักงานอีกคนคว้าหูหม้อไว้ได้ทัน น้ำซุปกระฉอกนิดหน่อยแต่ไม่โดนใคร |
### K5. ตะหลิววางบนขอบกระทะแบบไม่มั่นคง
#### คำตอบ
| รายการ    | รายละเอียด |
| --------- | ---------- |
| Danger    | ตะหลิวมีโอกาสร่วงลงมาโดนตัวพนักงาน หรือด้ามจับไปโดนสิ่งอื่น ตอนที่พนักงานหันไปหยิบเครื่องปรุงอย่างเร่งรีบ เพราะแรงสั่นสะเทือนจากการผัดทำให้ตะหลิวเสียการทรงตัว |
| Accident  | น้ำมันร้อน ๆ ในกระทะเดือดจนทำด้ามตะหลิวสไลด์ร่วงลงมา น้ำมันที่ติดอยู่กระเด็นเข้าตาเชฟ ส่วนตะหลิวตกลงพื้นกระแทกถังขยะล้ม ข้าวของกระจัดกระจาย |
| Near Miss | ตะหลิวลื่นไถลหลุดจากขอบกระทะร่วงลงด้านล่าง แต่โชคดีที่พนักงานใส่รองเท้าบูทหนาและชักเท้าหลบทันพอดี ตะหลิวเลยตกกระทบพื้นปูนเฉย ๆ ไม่โดนใคร |
### K6. พนักงานต้องยืนผัดอาหารต่อเนื่อง 11 ชั่วโมง
#### คำตอบ
| รายการ    | รายละเอียด |
| --------- | ---------- |
| Danger    | พนักงานมีโอกาสหน้ามืดเป็นลมหรือกล้ามเนื้ออักเสบเฉียบพลัน ตอนที่ต้องเร่งทำออเดอร์ช่วงเย็น เพราะร่างกายล้าสะสมจากการไม่ได้นั่งพักเลยตลอดทั้งวัน |
| Accident  | พนักงานยืนผัดข้าวขาเริ่มชาและหมดแรง จู่ ๆ ก็เกิดตะคริวกินที่น่องขาอย่างรุนแรงจนทรงตัวไม่อยู่ ล้มพับลงไปกองกับพื้น แขนไปปัดโดนจานข้าวข้าง ๆ ตกแตกเสียหาย |

### K7. ทางเดินในครัวแคบจนเดินสวนกันแทบไม่ได้
#### คำตอบ
| รายการ    | รายละเอียด |
| --------- | ---------- |
| Danger    | พนักงานมีโอกาสเดินชนกันจนของร้อนหรือของมีคมหลุดมือ ตอนที่คนหนึ่งกำลังยกหม้อแกงไปเสิร์ฟและอีกคนกำลังเดินสวนกลับมาเก็บจานอย่างรีบร้อน |
| Accident  | พนักงานถือถาดแกงเผ็ดเดินสวนกับล้างจานในช่องแคบ ข้อศอกชนกันอย่างจัง ถาดแกงคว่ำหลุดมือ น้ำแกงราดใส่หลังเพื่อนร่วมงาน เสื้อผ้าเปื้อนและผิวหนังแดงแสบร้อน |
| Near Miss | พนักงานยกตะกร้าแก้วน้ำเดินสวนกับคนถือมีด ปลายมีดเกือบจะปักเข้าที่ท้อง แต่คนถือมีดเอี้ยวตัวหลบและยกมีดขึ้นสูงแนบอกทันท่วงที เลยแค่เบียดกันนิดหน่อย |
### K8. พนักงานใช้ผ้าขี้ริ้วเก่าเช็ดพื้นจนพื้นยังลื่นอยู่
#### คำตอบ
| รายการ    | รายละเอียด |
| --------- | ---------- |
| Danger    | พนักงานมีโอกาสลื่นล้มหัวกระแทกตอนเดินรีบเอาวัตถุดิบไปส่งที่เตา เพราะผ้าเก่าบดบังและกระจายคราบมันทั่วพื้น แทนที่จะเช็ดคราบน้ำมันออกไปจริงๆ |
| Accident  | พนักงานเดินอุ้มลังหัวหอมผ่านจุดที่เพิ่งเช็ดเสร็จ เท้าเหยียบคราบลื่นที่ยังหลงเหลืออยู่จนหงายหลังล้มก้นกระแทกพื้นปูนอย่างแรง ลังแตกหัวหอมกลิ้งกระจายทั่วครัว |
| Near Miss | พนักงานเดินมาเจอจุดพื้นลื่นแล้วเท้าไถลวูบ คล้ายจะเสียหลักล้ม แต่พอดีมือไปคว้าจับขอบซิงค์ล้างจานสแตนเลสไว้มั่น เลยยังทรงตัวยืนอยู่ได้ |

### F13. พัดลมระบายอากาศเสีย ทำให้ความร้อนสะสมในโรงงาน
#### คำตอบ
| รายการ    | รายละเอียด |
| --------- | ---------- |
| Danger    | พนักงานมีโอกาสเป็นโรคลมแดด (Heatstroke) หรือขาดน้ำจนช็อก ตอนที่ต้องเดินคุมเครื่องจักรในโรงงานปิดช่วงบ่ายที่อบอ้าวและไม่มีอากาศถ่ายเท |
| Accident  | อุณหภูมิในโรงงานสูงจัดจนอบอ้าวเหมือนตู้อบ พนักงานที่ยืนคุมหน้าเตาหลอมเหงื่อท่วมตัว จู่ ๆ ร่างกายระบายความร้อนไม่ทัน เกิดอาการหน้ามืดหมดสติร่วงลงไปกองกับพื้นปูน |
| Near Miss | พนักงานอึดอัดหายใจไม่ออกขมับเต้นตุบ ๆ สัญญาณเตือนฮีทสโตรกเริ่มมา เกือบจะวูบคาเครื่องจักร แต่โชคดีที่เพื่อนสังเกตเห็นหน้าซีดเผือดเลยส่งน้ำเย็นให้ดื่มแล้วพาไปนั่งหน้าพัดลมไอเย็นทัน |

### C2. นั่งร้านไม่มีราวกันตก
#### คำตอบ
| รายการ    | รายละเอียด |
| --------- | ---------- |
| Danger    | คนงานมีโอกาสก้าวพลาดหลุดขอบนั่งร้านร่วงลงมาด้านล่าง ตอนที่ต้องแบกถังปูนขึ้นไปฉาบผนังตึกชั้น 3 ซึ่งเป็นพื้นที่สูงและลมแรง |
| Accident  | คนงานกำลังเอื้อมมือไปรับแผ่นไม้จากด้านล่าง จังหวะนั้นแผ่นไม้หนักจนดึงตัวคนงานเสียหลักลื่นไถลหลุดขอบนั่งร้านที่ไม่มีราวกันตก ร่วงลงกระแทกพื้นดินด้านล่าง ขาหักสองข้าง |
| Near Miss | คนงานก้าวถอยหลังเพื่อหลบสายไฟ แขนเสื้อเกือบจะเลยขอบนั่งร้านหล่นลงไปด้านล่าง แต่โชคดีที่ตะขอสายเซฟตี้เกี่ยวล็อกไว้กับโครงเหล็กแน่นหนา เลยแค่ห้อยโตงเตงอยู่กลางอากาศ |
### C3. บันไดเหล็กขึ้นชั้นสองไม่มีการยึดกับพื้น
#### คำตอบ
| รายการ    | รายละเอียด |
| --------- | ---------- |
| Danger    | คนงานมีโอกาสโดนบันไดสไลด์พาล้มกระแทกพื้น ตอนที่ต้องแบกเครื่องมือช่างหนัก ๆ เดินไต่ขึ้นบันไดเพื่อไปทำงานต่อบนชั้นสอง |
| Accident  | คนงานเดินขึ้นบันไดเหล็กไปได้ครึ่งทาง แรงกดและน้ำหนักตัวทำให้ฐานบันไดด้านล่างลื่นสไลด์หนีออกจากผนัง บันไดพับครูดลงมากับพื้น คนงานล้มกระแทกพื้นปูน หัวเข่าแตก |
| Near Miss | บันไดเหล็กเริ่มขยับและเอนเอียงคล้ายจะล้มลงมาตอนคนงานก้าวขึ้นขั้นแรก แต่โชคดีเพื่อนคนงานอีกสองคนยืนคุมอยู่ข้างล่างช่วยกันเข้าไปกอดและยึดขาบันไดไว้มั่น บันไดเลยไม่ล้ม |
### C4. คนงานเดินใกล้หลุมขุดดินที่ไม่มีแผงกั้น
#### คำตอบ
| รายการ    | รายละเอียด |
| --------- | ---------- |
| Danger    | คนงานมีโอกาสก้าวพลาดตกหลุมลึกหรือโดนดินปากหลุมถล่มทับ ตอนที่เดินแบกเหล็กเส้นผ่านขอบหลุมในช่วงพลบค่ำที่แสงสว่างหน้างานเริ่มไม่พอ |
| Accident  | คนงานเดินคุยงานกันเพลิน ๆ ก้าวเท้าเหยียบดินปากหลุม ดินเกิดทรุดตัวทำร่างคนงานร่วงหล่นลงไปในหลุมลึก 3 เมตร กระแทกเศษหินก้นหลุมจนข้อเท้าพลิกอักเสบรุนแรง |
| Near Miss | คนงานเดินแบกของลื่นดินทรายตรงขอบหลุม ตัวถล่าวูบเกือบจะร่วงลงไปด้านล่าง แต่โชคดีที่หันไปใช้มือคว้าท่อเหล็กที่ปักอยู่ข้าง ๆ ทัน เลยทรงตัวยืนนิ่งอยู่ริมขอบหลุมได้ |
### C5. เครื่องตัดเหล็กวางบนพื้นโดยไม่ถอดปลั๊ก
#### คำตอบ
| รายการ    | รายละเอียด |
| --------- | ---------- |
| Danger    | คนงานมีโอกาสโดนใบมีดตัดหรือสับนิ้วมือขาด ตอนที่เดินข้ามหรือเอื้อมมือไปหยิบเศษเหล็กแถวนั้น แล้วบังเอิญไปเตะหรือกดโดนสวิตช์เครื่องที่ยังสตาร์ทค้างไว้ |
| Accident  | คนงานลากแผ่นเหล็กผ่านเครื่องตัด สายไฟแผ่นเหล็กไปเกี่ยวโดนไกสวิตช์ของเครื่องตัดเหล็กที่ไม่ได้ถอดปลั๊ก เครื่องทำงานทันที ใบตัดเหวี่ยงตัดผ่านรองเท้าผ้าใบจนนิ้วเท้าขาด |
| Near Miss | มีคนเผลอไปเหยียบโดนสวิตช์เครื่องตัดเหล็กจนเครื่องหมุนเสียงดังลั่น แต่อุปกรณ์ล็อกใบมีด (Safety Guard) ของเครื่องยังทำงานได้ดี ใบมีดเลยไม่กระเด็นไปโดนใคร |
### C6. สายไฟต่อพ่วงหลายชั้นแบบพาดไปมาบนพื้น

| Near Miss | คนงานเดินสะดุดขยุ้มสายไฟพ่วงจนตัวเซหวุดหวิดจะล้มหน้าคว่ำ แต่โชคดีที่เอามือไปเกาะรถเครนที่จอดนิ่งอยู่ข้าง ๆ ทัน เลยยืนตั้งหลักได้และไม่โดนไฟดูดเพราะใส่บูทยาง |
### C7. คนงานยกถุงปูนหนักโดยไม่มีอุปกรณ์ช่วยยก
#### คำตอบ
| รายการ    | รายละเอียด |
| --------- | ---------- |
| Danger    | คนงานมีโอกาสเกิดอาการกล้ามเนื้อหลังส่วนล่างอักเสบเฉียบพลันหรือหมอนรองกระดูกทับเส้นประสาท ตอนที่ต้องก้มลงยกถุงปูนหนัก 50 กิโลกรัมข้ามคานปูนอย่างต่อเนื่อง |
| Accident  | คนงานก้มหลังงอยกถุงปูนขึ้นจากพื้น จังหวะที่พยายามเหวี่ยงถุงปูนขึ้นบ่า เกิดเสียงดังกึดที่บั้นท้าย กล้ามเนื้อหลังฉีกขาดฉับพลัน ล้มพับร้องลั่นด้วยความเจ็บปวดลุกไม่ขึ้น |
| Near Miss | คนงานยกถุงปูนขึ้นมาแล้วรู้สึกแปล๊บที่เอวคล้ายหลังจะเดาะ เกือบจะฝืนยกต่อจนบาดเจ็บ แต่ฉุกคิดได้เลยรีบปล่อยถุงปูนลงพื้นทันที แล้วเปลี่ยนไปเรียกเพื่อนอีกคนมาช่วยประคองยก |

| Near Miss | คนงานเดินลุยโคลนแล้วเท้าไถลพรืดเกือบจะล้มหัวคะมำไปข้างหน้า แต่โชคดีที่มือขวาไปคว้าจับเสาเข็มคอนกรีตที่ปักแน่นอยู่ข้างทางไว้ได้ทัน ทรงตัวอยู่รอดไปได้ |
### C9. เศษเหล็กยาววางพาดทางเดิน

| Danger    |พนักงานมีโอกาสเกิดอุบัติเหตุศีรษะกระแทกพื้นปูนอย่างรุนแรงจนถึงแก่ชีวิต ตอนที่ขี่รถมอเตอร์ไซค์ไปส่งเอกสารระหว่างอาคารแล้วเจอทรายหรือน้ำมันลื่นบนพื้นลานจอด|
| Accident  |พนักงานขี่มอเตอร์ไซค์เลี้ยวโค้งผ่านลานโหลดรถ ล้อหน้าเหยียบเศษกรวดทรายทำให้รถเสียหลักล้มคว่ำไถลไปกับพื้น ร่างพนักงานหลุดจากรถ หัวกระแทกพื้นปูนเต็ม ๆ จนสลบคาที่และเลือดอาบหัว|
| Near Miss |มอเตอร์ไซค์สะดุดหลุมบนพื้นลานจอดจนรถสะบัดโคลงเคลง เกือบจะล้มหัวฟาดพื้นปูนอยู่แล้ว แต่โชคดีที่พนักงานใช้เท้าข้างหนึ่งยันพื้นช่วยประคองรถไว้ได้ทัน รถเลยไม่ล้มและทรงตัวขี่ต่อได้|
