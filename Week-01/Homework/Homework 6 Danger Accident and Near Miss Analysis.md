## Homework: วิเคราะห์ Danger / Accident / Near Miss
จาก Hazard ที่กำหนด ให้คุณวิเคราะห์ 3 มิติ ได้แก่
- Danger (อันตรายที่อาจเกิดขึ้น)
- Accident (เหตุการณ์ที่เกิดขึ้นจริง)
- Near Miss (เหตุเฉียดที่เกือบเกิด แต่ยังไม่เกิด)
ให้เขียนลงในตารางที่กำหนด โดยใช้ภาษาของตัวเอง ห้ามท่องนิยาม ห้ามคัดลอกทฤษฎี ให้เขียนแบบภาษาคนทั่วไป เหมือนเล่าเหตุการณ์ให้เพื่อนฟัง
สามารถส่งตรวจคราวเดียวทั้งหมดหรือครั้งละหมวดก็ได้
คุณสามารถส่งคำตอบใหม่ได้เรื่อยๆ จนกว่าจะพอใจคะแนน TA AI จะตรวจให้ทุกครั้งแบบเข้มงวด

## แนวทางการเขียน Danger, Accident และ Near Miss
### 👉 วิธีเขียน Danger ให้ถูกต้อง
Danger = “โอกาสที่จะเกิดเหตุ” ต้องมี **สถานการณ์เฉพาะเจาะจง** เช่น
- ตอนเดินถือของร้อน
- ตอนเดินสวนกันในครัวแคบ
- ตอนรีบทำงาน
ห้ามเขียนว่า “มีโอกาสลื่นล้ม” เฉย ๆ เพราะมันสั้นเกินไป ต้องระบุสถานการณ์ที่เจาะจงด้วย

### 👉 วิธีเขียน Accident ให้ถูกต้อง
Accident = “เหตุการณ์จริงที่เกิดขึ้นแล้ว” ต้องมี **ลำดับเหตุการณ์** เช่น
1. เหยียบน้ำมัน
2. ลื่นล้ม
3. มือกระแทกกระทะร้อน
4. อาหารหก
ห้ามเขียนว่า “ลื่นล้มเพราะเดินไม่ระวัง” เพราะเป็นการโทษคน และไม่ใช่หลัก OHS

### 👉 วิธีเขียน Near Miss ให้ถูกต้อง
Near Miss = “เกือบเกิด แต่ไม่เกิด” ต้องมี **สิ่งที่ช่วยให้ไม่เกิดเหตุ** เช่น
- จับโต๊ะทัน
- ยกเท้าหลบทัน
- เพื่อนดึงตัวไว้
ห้ามเขียนว่า “เกือบล้ม” เฉย ๆ ต้องบอกว่า “เพราะอะไรถึงไม่ล้ม”

### K1. พื้นครัวมันลื่นเพราะน้ำมันกระเด็นตอนผัดไฟแรง
#### คำตอบ (ตัวอย่าง)

| รายการ    | รายละเอียด                                                                                                                                    |
| --------- | --------------------------------------------------------------------------------------------------------------------------------------------- |
| Danger    | พนักงานมีโอกาสลื่นล้มตอนเดินถือจานร้อน โดยเฉพาะช่วงที่ครัวแคบและต้องเดินสวนกัน เพราะพื้นมีน้ำมันเคลือบอยู่ทำให้เท้าไถลง่าย                    |
| Accident  | พนักงานเดินถือชามร้อนแล้วเหยียบน้ำมัน ทำให้ลื่นล้ม มือกระแทกกระทะร้อนจนพอง และอาหารหกกระจายเต็มพื้น เกิดความเสียหายทั้งต่อร่างกายและงานในครัว |
| Near Miss | พนักงานเดินผ่านจุดที่มีน้ำมันแล้วรู้สึกเท้าไถล แต่จับโต๊ะทันเลยไม่ล้ม เหตุการณ์นี้เกือบจะกลายเป็นอุบัติเหตุจริง แต่โชคดีที่ควบคุมตัวเองได้ทัน |

### K2. มีดวางคว่ำปลายชี้ออกนอกโต๊ะใกล้ทางเดิน
#### คำตอบ
| รายการ    | รายละเอียด |
| --------- | ---------- |
| Danger    | พนักงานเสิร์ฟเสี่ยงที่จะโดนปลายมีดเกี่ยวหรือปักเข้าที่หน้าขาหรือสะโพก ตอนที่กำลังเดินเร่งรีบถือถาดอาหารผ่านช่องทางเดินแคบๆ นั้น        |
| Accident  | พนักงานเดินเบียดทางเดินเข้าไปหยิบของ แล้วสะโพกไปเกี่ยวโดนด้ามมีด ทำให้มีดหมุนคว่ำตกลงมาปักใส่หลังเท้าเต็มๆ จนแผลลึกและเลือดไหลนองพื้นครัว          |
| Near Miss |พนักงานเดินเอียงตัวหลบสิ่งกีดขวางแล้วชายเสื้อไปปัดโดนมีดจนเกือบจะร่วงลงมา แต่โชคดีที่พนักงานล้างจานตาไว ยื่นมือมาคว้าด้ามมีดกลางอากาศไว้ทัน มีดเลยไม่ปักเท้าใคร         |
### K3. ถังแก๊สตั้งชิดเตาไฟจนผิวถังร้อน
#### คำตอบ
| รายการ    | รายละเอียด |
| --------- | ---------- |
| Danger    | ความเสี่ยงที่ตัวถังและสายส่งแก๊สจะสะสมความร้อนสูงจนละลายหรือระเบิดพรึบขึ้นมา ในจังหวะที่ป้าจำปีเร่งไฟทำอาหารชุดใหญ่ช่วงลูกค้าแน่นร้าน           |
| Accident  | ป้าจำปีเร่งไฟผัดกะเพรานานจนความร้อนสะสมทำให้สายยางแก๊สข้างเตาละลาย แก๊สรั่วพุ่งออกมาเจอเปลวไฟจนเกิดไฟลุกพรึบท่วมถังแก๊สทันที ทำให้พนักงานหน้าเตาโดนไฟลวกแขนและผมไหม้เกรียม|
| Near Miss |  ป้าจำปีผัดไฟลุกท่วมแล้วสะเก็ดไฟพุ่งไปโดนตัวถังแก๊สจนเริ่มมีกลิ่นไหม้ แต่โชคดีที่พนักงานเสิร์ฟหันไปเห็นทัน เลยรีบเอาแผ่นสแตนเลสข้างๆ มาเสียบกั้นบังความร้อนไว้ได้หวุดหวิด ถังแก๊สเลยยังไม่ระเบิด |
### K4. หม้อร้อนตั้งบนโต๊ะที่โยกเยก
#### คำตอบ
| รายการ    | รายละเอียด |
| --------- | ---------- |
| Danger    | โอกาสที่หม้อน้ำซุปเดือดๆ จะไถลลื่นหรือคว่ำเทกระจาดลงมา ตอนที่พนักงานกระแทกจานลงบนโต๊ะ หรือเดินสะดุดขาโต๊ะที่กำลังโยกเยกอยู่           |
| Accident  | พนักงานล้างจานยกกองจานมาวางกระแทกบนโต๊ะเตรียมของ ทำให้โต๊ะที่ขาไม่เท่ากันโยกอย่างแรง ส่งผลให้หม้อน้ำแกงร้อนๆ ที่ตั้งอยู่ข้างบนไถลร่วงลงมาลาดใส่ขาพนักงาน ลวกจนผิวหนังพองพุพองแดงก่ำ|
| Near Miss | มีคนเดินชนมุมโต๊ะจนหม้อน้ำซุปเอียงวืดทำท่าจะคว่ำลงมา แต่โชคดีที่พนักงานอีกคนยืนอยู่ข้างๆ มือไว ตะปบหูหม้อสองข้างดึงกลับมาตั้งตรงไว้ได้ทัน น้ำซุปเลยไม่หกมาลวกใคร|

#### คำตอบ

| Near Miss |  พนักงานลนลานวิ่งไปวางมีดทำครัวไว้บนขอบตู้เย็น มีดสไลด์ทำท่าจะหล่นลงมาสับหัวพนักงานล้างจานที่กำลังก้มเก็บของอยู่ ดีที่เพื่อนร่วมงานตะโกนบอก “ระวังมีด!” พนักงานเลยโดดหลบทัน มีดเลยสับลงพื้นแทน          |

#  **หมวด 2 — โรงงาน / การผลิต (13 ข้อ)**
### F1. พื้นโรงงานมีเศษน้ำมันเครื่องหยดเป็นทางยาว
#### คำตอบ (ตัวอย่าง)

| รายการ    | รายละเอียด                                                                                         |
| --------- | -------------------------------------------------------------------------------------------------- |
| Danger    | พนักงานมีโอกาสลื่นล้มตอนเดินขนชิ้นงาน หรืออาจสะดุดลื่นไปชนเครื่องจักรที่กำลังทำงานอยู่             |

| Near Miss | พนักงานเหยียบน้ำมันแล้วเท้าไถล แต่จับราวเครื่องจักรไว้ทัน เลยไม่ล้ม                                |
### F2. เครื่องจักรไม่มีฝาครอบป้องกันชิ้นส่วนหมุน

| Accident  |พนักงานก้มลงไปเช็กระดับชิ้นงาน ปลายเสื้อช็อปที่ไม่ได้ยัดใส่กางเกงถูกแกนหมุนเหวี่ยงเกี่ยวเข้าไปอย่างรวดเร็ว เครื่องจักรดึงร่างพนักงานกระแทกเข้ากับแท่นเครื่องจนซี่โครงหักและแขนบิดผิดรูป            |
| Near Miss |พนักงานก้มไปดูเครื่องแล้วปลายผมเกือบจะโดนเฟืองหมุนม้วนจับ แต่โชคดีที่หมวกนิรภัยที่ใส่ไว้สกัดแกนหมุนตัดขวางจนเครื่องขัดกึกหยุดทำงานทัน รอดจากการถูกถลกหนังหัวไปได้อย่างฉิวเฉียด            |
### F3. สายพานลำเลียงไม่มีรั้วกั้นด้านข้าง
#### คำตอบ
| รายการ    | รายละเอียด |
| --------- | ---------- |
| Danger    | โอกาสที่ชิ้นงานหนักๆ จะสไลด์ตกขอบสายพานลงมาทับเท้า หรือพนักงานเอื้อมมือไปหยิบของผิดท่าแล้วมือหลุดเข้าไปติดในช่องว่างสายพาน           |
| Accident  |  ชิ้นงานเหล็กหล่อเกิดค้างและเบียดกันบนสายพานจนเสียสมดุล ดันกันเองร่วงตกขอบสายพานทับหลังเท้าพนักงานที่ยืนคุมเครื่องอยู่ด้านล่างจนกระดูกแตก นิ้วเท้าหัก 3 นิ้ว          |
| Near Miss |  ชิ้นงานพลาสติกขนาดใหญ่เหวี่ยงหลุดจากสายพานทำท่าจะร่วงใส่หัวพนักงานที่เดินผ่านมาพอดี ดีที่หัวหน้างานยืนคุมอยู่ฝั่งตรงข้ามตะโกนลั่นพร้อมผลักผลักพนักงานคนนั้นออกทัน ชิ้นงานเลยตกกระแทกพื้นเปล่าๆ          |
### F4. พนักงานต้องยกชิ้นงานหนักซ้ำ ๆ ตลอดกะ
#### คำตอบ
| รายการ    | รายละเอียด |
| --------- | ---------- |
| Danger    | ความเสี่ยงที่จะเกิดภาวะกล้ามเนื้อหลังส่วนล่างฉีกขาดสะสม หรือหมอนรองกระดูกทับเส้นประสาทจากการเคลื่อนไหวผิดหลักการยศาสตร์ต่อเนื่องหลายชั่วโมง           |
| Accident  | พนักงานยกกล่องเหล็กหนัก 20 กิโลกรัมกล่องที่ร้อยของวัน จังหวะเอี้ยวตัววางชิ้นงาน หลังเกิดเสียงลั่น “ป๊อก” พร้อมความปวดเจ็บเจียนตาย พนักงานล้มฟุบคาแท่นงาน ขยับช่วงล่างไม่ได้เนื่องจากหมอนรองกระดูกเคลื่อนเฉียบพลัน           |
| Near Miss | พนักงานเอื้อมยกกล่องชิ้นงานหนักแล้วรู้สึกแปล๊บวาบที่แผ่นหลังบั้นเอว เลยรีบปล่อยกล่องลงบนโต๊ะทันที เกือบหลังหักคาที่ ได้พักดื่มน้ำยืดเส้นยืดสายจนอาการดีขึ้น           |

| Accident  | กล่องวัตถุดิบที่วางซ้อนสูงลิ่วเอียงกะเท่เร่และล้มครืนถล่มลงมาทับพนักงานที่กำลังก้มเช็กสต็อกอยู่ด้านล่าง กล่องหนักกระแทกหัวและหลังจนสลบคาที่และกระดูกไหปลาร้าหัก           |
| Near Miss | กองกล่องด้านบนขยับตัวเลื่อนสไลด์ร่วงลงมาเสียงดังโครมลั่นคลังสินค้า โชคดีพนักงานหูไวได้ยินเสียงกระดาษคราฟท์ฉีกขาดก่อนเลยกระโดดหลบพุ่งตัวออกข้างทัน กล่องเหล่านั้นเลยตกลงมาบี้พาเลทเปล่าแทน           |
### F6. พนักงานเดินผ่านโซนยกของด้วยเครนโดยไม่มีเส้นแบ่งเขต

### F7. ปุ่ม emergency stop อยู่ไกลจากตำแหน่งทำงานจริง

| Accident  | ถุงมือของพนักงานเข้าไปติดในลูกกลิ้งบดชิ้นงาน พนักงานพยายามเอื้อมมืออีกข้างไปกดปุ่มฉุกเฉินแต่เอื้อมไม่ถึงเพราะอยู่ไกลเกินกว่า 3 เมตร กว่าเพื่อนร่วมงานจะวิ่งมาช่วยกดปิด นิ้วมือก็ถูกบดละเอียดไปแล้ว 2 นิ้ว           |
| Near Miss |  ชิ้นงานเหล็กเกิดติดขัดในห้องเกียร์จนเครื่องสั่นสะเทือนทำท่าจะระเบิด พนักงานรีบวิ่งสุดฝีเท้าไปกดปุ่มหยุดฉุกเฉินที่อยู่อีกฝั่งของห้อง ทันเวลาก่อนที่ฟันเฟืองจะแตกกระจายออกมาเซฟเครื่องจักรไว้ได้ฉิวเฉียด        |

### F10. พนักงานทำงานใกล้สารเคมีโดยไม่มีหน้ากาก

### T11. รถบรรทุกจอดบนพื้นเอียงโดยไม่ใช้ตัวหนุนล้อ
#### คำตอบ
| รายการ    | รายละเอียด |
| --------- | ---------- |
| Danger    |   ความเสี่ยงที่ระบบเบรกมือของรถจะลื่นไถลหรือขัดข้อง ทำให้รถบรรทุกหนักไหลถอยหลังลงมาชนอาคารคลังสินค้าหรือทับคนงานด้านล่าง         |
| Accident  |   รถสิบล้อจอดบนทางลาดระบบเบรกมือล้าหลุดวืด รถไหลถอยหลังลงมาด้วยความเร็วพุ่งชนถล่มอัดพนักงานที่กำลังยืนเช็กของอยู่ท้ายตู้ติดเข้ากับผนังโกดังแหลกละเอียด เสียชีวิตทันทีในที่เกิดเหตุ         |
| Near Miss |  รถเทรลเลอร์จอดติดเครื่องบนทางลาดแล้วตัวรถเริ่มสไลด์ไหลถอยหลังกะทันหัน เกือบจะไหลไปทับพนักงานล้างรถ โชคดีคนขับตาไวโดดขึ้นรถเหยียบเบรกเท้าซ้ำมิดได้ทันเวลา รถหยุดห่างจากคนไปคืบเดียว          |
### T12. พนักงานขี่มอเตอร์ไซค์ในลานขนส่งโดยไม่ใส่หมวกกันน็อก
#### คำตอบ
| รายการ    | รายละเอียด |
| --------- | ---------- |
| Danger    |   เสี่ยงที่จะเกิดอุบัติเหตุรถเฉี่ยวชน ล้มคว่ำ แล้วหัวพนักงานกระแทกเข้ากับพื้นปูนหรือขอบล้อรถใหญ่โดยตรงไม่มีอะไรป้องกัน         |
| Accident  | พนักงานขี่มอเตอร์ไซค์ส่งบิลในลานลื่นสะดุดกรวดหินรถล้มไถล ร่างกิ้งเด้ง หัวพนักงานกระแทกเข้ากับขอบแท่นปูนซีเมนต์อย่างจังจนกะโหลกร้าว เลือดออกในสมอง บาดเจ็บสาหัสเป็นตายเท่ากัน           |
| Near Miss |   พนักงานขี่มอเตอร์ไซค์หักหลบรถเข็นปูนจนรถเสียหลักสะบัดล้มคว่ำ ตัวพนักงานไถลครูดไปกับพื้นเกือบเอาหัวพุ่งเข้าใต้ล้อรถบรรทุกที่กำลังวิ่ง โชคดีเอามือยันพื้นดันตัวเบรคชะลอแรงไว้ทัน หัวเลยหยุดก่อนถึงล้อรถไปนิดเดียว         |
